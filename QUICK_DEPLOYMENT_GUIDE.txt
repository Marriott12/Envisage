STEP 3: CONFIGURE OAUTH APPS
-----------------------------
Google OAuth:
1. Go to: https://console.cloud.google.com
2. Create OAuth 2.0 credentials
3. Authorized redirect URIs: https://envisagezm.com/api/auth/google/callback
4. Add credentials to backend .env:
   GOOGLE_CLIENT_ID=xxxxx
   GOOGLE_CLIENT_SECRET=xxxxx

Facebook OAuth:
1. Go to: https://developers.facebook.com
2. Create app, get App ID and Secret
3. Add redirect URI: https://envisagezm.com/api/auth/facebook/callback
4. Add credentials to backend .env:
   FACEBOOK_CLIENT_ID=xxxxx
   FACEBOOK_CLIENT_SECRET=xxxxx
